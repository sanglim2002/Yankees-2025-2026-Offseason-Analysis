# Yankees-2025-2026-Offseason-Analysis
## 
This case study by Samuel Anglim is a "Track B" capstone project for the Google Professional Data Analytics certificate. It details how the New York Yankees Baseball Club should construct its 2026 roster using a cost- and trade-effective analytical approach. 

---

## Phase 1: Ask

* **Business Task:** To conduct a comprehensive financial and statistical analysis of the 2025 Yankees, benchmarking payroll efficiency and on-field performance against a 5-10 year historical model of recent successful championship-winning teams. The final deliverable is a set of specific, cost-effective recommendations for free agents and trades to address the team's statistical imbalances.
* **Stakeholders:** Hal Steinbrenner (Owner), Brian Cashman (General Manager), Aaron Boone (Manager), and the Scouting Department.
